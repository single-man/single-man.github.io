一、排序算法

1. 冒泡排序

原理

冒泡排序(Bubble Sort)是一种简单的排序算法，它通过一次比较两个相邻的元素，看两个元素是否满足大小关系要求，如果不满足则交换两个元素。每一次冒泡都会至少产生一次数据移动，直到将最小的元素“浮”到数列顶端

算法的过程描述：

  • 比较相邻的元素。如果第一个比第二个大，就交换它们两个；
  • 对每一对相邻元素做同样的工作，从开始的第一对到结尾的最后一对，这样在最后的元素就是最大的元素；
  • 针对所有元素重复以上两步骤，除了最后一个；
  • 重复前三步，直到排序完成；

示意图

代码实现

public void bubbleSort(int[] array){
    int length=array.length;//待排序元素的个数
    if(length<=1){
        return;
    }
    //冒泡排序
    for(int i=0;i<length;i++){//控制冒泡排序的次数
        for(int j=0;j<length-i-1;j++){//相邻元素两两比较
            //如果前一个数据大于后一个数据则进行交换
            if(array[j]>array[j+1]){
               int temp=array[j];
               array[j]=arry[j+1];
               array[j+1]=temp;
           }
        }
    } 
}

2. 插入排序

原理

插入排序（Insertion Sort）的原理是：将数组中的数据分成两个区域，已排序区域和未排序区域。初始时已排序区域只有一个元素，就是数组的第一个元素。插入算法的核心思想是取未排序区间中的元素，在已排序的区域中找到合适的插入位置将其插入，并保证已排序区间数据一直有序。重复这个过程，直到未排序区间元素为空，算法结束。

算法描述过程如下

  • 从第一个元素开始，该元素可以认为已经被排序；
  • 取出下一个元素，在已经排序的序列中从后向前扫描；
  • 如果该元素（已排序）大于新元素，将已排序的元素继续向前找；
  • 重复步骤3，直到找到已排序的元素小于或者等于新元素的位置；
  • 将新元素查到该位置后面；
  • 重复2-5

示意图

代码实现

public void insertionSort(int[] array){
    int len=array.length;
    if(len<=1){
        return;
    }
    for(int i=1;i<len;i++){
        //取出当前序列中未排序的元素，及当前要和已排序比较的元素
        int current=array[i];
        //定义在有序区间从后往前扫描的指针
        int preIndex=i-1;
        while(preIndex>=0 && array[preindex]>current){
            arr[preIndex+1]=arr[preIndex];
            preIndex--;
        }
        array[preIndex+1]=current
    }
}

3. 选择排序

原理

选择排序（Selection Sort）的原理有点类似插入排序，也分已排序区间和未排序区间。但是选择排序每次都会从未排序区间中找到最小的元素，将其放到已排序区间的末尾。

算法的描述过程

  • 初始状态：无序区间为R[1..n],有序区间为空；
  • 第i趟排序 ( i = 1,2,3,…n-1) 开始时，当前有序区和无序区分别为 R[1…i-1] 和 R[i..n] 。该趟排序从当前无序区中选出关键字最小的记录R[k]，将它与无序区的第一个记录交换，使R[1..i]和R[i+1..n]分别变为记录个数增加1个的新有序区和记录个数减少1个无序区；
  • n-1趟结束，数组有序化了。

示意图

代码实现

public void selectionSort(int[] arr){
	int len=arr.length;
    if(len<=1){
        return;
    }
    
    for(int i=0;i<len;i++){
        int minIndex=i;
        //从未排序的序列中找到最小值	
        for(int j=i;j<len;j++){
            if(arr[minIndex]>arr[j]){
                minIndex=j
            }
        }
        int current=arr[i];
        arr[i]=arr[minIndex];
        arr[minIndex]=current;
    }
    
}

4. 归并排序

原理

归并排序（Merge Sort）使用的是分治思想，而分治方法又用到了递归思想来实现。分治是一种解决问题的处理思想，递归是一种编程技巧，这两者并不冲突。而对于递归就是找到递推公式和终止条件，所以我们可以先写出归并排序的递推公式

mergeSort(m->n) = mergeSort(mergeSort(m->k),mergeSort(k+1->n));当m=n时终止

算法描述过程

  • 把长度为n的输入序列分成两个长度为n/2的子序列；

  • 对这连个子序列分别采用归并排序；

  • 将两个排序好的子序列合并成一个最终的排序序列

    示意图

代码实现

/**
     * 归并排序
     *	将数列分解
     */
    public static int[] MergeSort(int[] array) {
        if (array.length < 2) return array;
        int mid = array.length / 2;
        int[] left = Arrays.copyOfRange(array, 0, mid);
        int[] right = Arrays.copyOfRange(array, mid, array.length);
        return merge(MergeSort(left), MergeSort(right));
    }
    /**
     * 归并排序——将两段排序好的数组结合成一个排序数组
     * 将分解后的数列进行合并
     */
    public static int[] merge(int[] left, int[] right) {
        //定义一个result数组 存放合并后的数据
        int[] result = new int[left.length + right.length];
        //index是result数组的索引 i是前半段数组索引 j是后半段数组索引
        for (int index = 0, i = 0, j = 0; index < result.length; index++) {
            if (i >= left.length)//此时说明左半段数组已完全放入result数组中
                result[index] = right[j++];//把右半段数组直接放入rusult中，无需比较
            else if (j >= right.length)//此时说明右半段数组先放入result中
                result[index] = left[i++];
            else if (left[i] > right[j])//左半段第一个数大于右半段第一个数
                result[index] = right[j++];//将右半段第一个数放入result数组中
            else
                result[index] = left[i++];
        }
        return result;
    }

5. 快速排序

原理

快速排序（Quick Sort）算法，简称快排，利用的是分治思想；快排的思路：如果要对m->n之间的数列进行排序，我们选择m->n之间的任意一个元素数据作为分区点(Pivot)，然后我们遍历m->n之间的所以元素，将小于pivot的元素放到左边，大于pivot的元素放到右边，pivot放到之间，这样整个数列就被分成三个部分，m-k-1之间的元素是小于pivot的，之间是pivot，k+1->m之间的元素是大于pivot的。然后再根据分治递归的思想处理两边区间的元素数列，直到区间缩减为1，就说明整个序列已经有序了

算法描述过程

  • 从数列中挑出一个元素，称为“基准”（pivot）；
  • 重新排序数列，所有比基准小的元素摆放在基准前面，所有元素比基准大的摆放在基准后面（相同的数可以到任一边）。在这分区退出之后，该基准就处于数列的中间位置；
  • 递归地把小于基准值的元素子序列和大于基准值的元素子序列排序

示意图

代码实现

/**
     * 快速排序方法
     */
    public static int[] QuickSort(int[] array, int start, int end) {
        if (array.length < 1 || start < 0 || end >= array.length || start > end) 
            return null;
        int smallIndex = partition(array, start, end);
        if (smallIndex > start)
            QuickSort(array, start, smallIndex - 1);
        if (smallIndex < end)
            QuickSort(array, smallIndex + 1, end);
        return array;
    }
    /**
     * 快速排序算法——partition
     * 找基准点
     */
    public static int partition(int[] array, int start, int end) {
        //将最后一个元素设为基准点
        int pivot = array[end];
        //int pivot = (int) (start + Math.random() * (end - start + 1));
        int smallIndex = start - 1;
        swap(array, pivot, end);
        for (int i = start; i <= end; i++)
            if (array[i] <= array[end]) {
                smallIndex++;
                if (i > smallIndex)
                    swap(array, i, smallIndex);
            }
        return smallIndex;
    }
 
    /**
     * 交换数组内两个元素
     */
    public static void swap(int[] array, int i, int j) {
        int temp = array[i];
        array[i] = array[j];
        array[j] = temp;
    }

6. 桶排序

原理

桶排序（Bucked Sort）是将待排序集合中处于同一值域的元素存入同一桶中，也就是根据元素值特性将集合拆分为多个区域，则拆分后形成的多个桶，从值域上开是处于有序状态的。对每个桶中元素进行排序，则所有桶中元素构成的集合是已排序的。

算法描述过程

  • 人为设置一个BucketSize，作为每个桶所能放置多少个不同数值（例如当BucketSize==5时，该桶可以存放｛1,2,3,4,5｝这几种数字，但是容量不限，即可以存放100个3）；

  • 遍历输入数据，并且把数据一个一个放到对应的桶里去；

  • 对每个不是空的桶进行排序，可以使用其它排序方法，也可以递归使用桶排序；

  • 从不是空的桶里把排好序的数据拼接起来。

    注意，如果递归使用桶排序为各个桶排序，则当桶数量为1时要手动减小BucketSize增加下一循环桶的数量，否则会陷入死循环，导致内存溢出。
